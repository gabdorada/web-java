# Web Java - Módulo 04 (TIC em Trilhas)

Repositório dedicado ao estudo e desenvolvimento de aplicações Web com Java e tecnologias Front-end.

## 📂 Projetos do Módulo

1. **01-landing-page-html-css**: Construção de interface estática focada em estruturação e estilização.
2. **02-consumo-api-fetch**: Aplicação prática de requisições assíncronas no Front-end utilizando Fetch API.
3. **03-projeto-crud**: Aplicação completa com operações de Create, Read, Update e Delete integrada ao backend Java.

## 🛠️ Tecnologias Utilizadas
- HTML5 / CSS3
- JavaScript (Fetch API)
- Java Web
