# ☕ Web Java - (TIC em Trilhas)

Repositório dedicado ao estudo e desenvolvimento de aplicações Web com Java e tecnologias Front-end.

---

## 📌 Estrutura do Repositório & Documentação

Para tornar o processo de aprendizado transparente e documentado, **cada subpasta possui o seu próprio arquivo `diario.md`**. Nesses diários você encontrará:
* **Notas de estudo:** insights, desafios e explicações do código.
* **Prints de resultados:** prévias visuais da aplicação em funcionamento.
* **Metodologia:** referências utilizadas e observações sobre o uso de ferramentas/IA.
